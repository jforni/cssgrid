CSS Grid

Descubriendo CSS Grid Layout

¡Basta de sufrir con la maquetación!

El "Antes" de Grid - ¿Por qué lo necesitamos?

Hasta hace no mucho, posicionar elementos en una página era una de las tareas más frustrantes de CSS. Los métodos antiguos como float o position eran complejos e impredecibles.

Llegó Flexbox, una mejora increíble, pero está diseñado para maquetar en una sola dimensión (una fila o una columna). Crear estructuras complejas de varias dimensiones podía volverse muy laborioso.

Precisábamos una herramienta más potente para crear estructuras de dos dimensiones (filas y columnas) de forma rápida y sencilla. Así nace CSS Grid.

¿Qué es CSS Grid? - El Concepto Clave

Piensen en Grid como una hoja de cálculo o un tablero de ajedrez directamente en su CSS. Es un sistema de maquetación bidimensional: nos permite controlar filas y columnas al mismo tiempo.

Toma la filosofía de Flexbox, por lo que muchos conceptos resultarán familiares. Además, tiene soporte total en todos los navegadores modernos desde 2017. ¡Úsenla sin miedo!

El Vocabulario de Grid

Para dominar Grid, primero tenemos que conocer sus partes:

Contenedor (Grid Container)
Ítem
Área (Grid Area)
Ítem
Celda
Ítem
  • Contenedor: El elemento padre que tendrá el `display: grid`.
  • Ítem: Cada uno de los hijos directos del contenedor.
  • Celda: La unidad más pequeña de la grilla.
  • Área: Un conjunto de celdas que forman un rectángulo.

¡Manos a la Obra! - Cómo activar Grid

Para empezar a usar Grid, solo necesitamos aplicar display: grid al elemento contenedor.

HTML Básico

<div class="grid"> <!-- Contenedor -->
  <div class="item">Item 1</div> <!-- Ítem -->
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

CSS Mágico

.grid {
  display: grid;
}

Ejemplos Funcionales

1. Grilla con Píxeles (Estática)

Creamos una grilla de 2x2 con tamaños fijos. Esta grilla no se adaptará al tamaño de la pantalla.

Código

.grid-container {
  display: grid;
  grid-template-columns: 50px 300px;
  grid-template-rows: 200px 75px;
}

2. La Unidad `fr` (Flexible)

La unidad fr (fracción) divide el espacio disponible proporcionalmente. ¡Prueba a redimensionar la ventana para ver su magia!

Código

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 2fr 1fr;
  height: 300px;
}

3. Función `repeat()` y Propiedad `gap`

Simplificamos la creación de múltiples columnas con repeat() y añadimos espacio entre celdas con gap.

Código

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 100px;
  gap: 10px;
}

Un Vistazo al Futuro

Hoy vimos las bases, ¡pero el universo de Grid es enorme! Aquí hay más conceptos por explorar:

minmax()

Define tamaños mínimos y máximos para las celdas, ¡ideal para responsive!

auto-fill / auto-fit

Permite al navegador decidir cuántas columnas caben en el espacio.

grid-template-areas

Permite "dibujar" el layout de forma visual en el CSS.

Atajos

Propiedades como `grid` y `grid-template` para escribir todo en una línea.

Resumen y Próximos Pasos

Lo que nos llevamos hoy

  • Grid es para maquetación en 2D (filas y columnas).
  • Todo empieza con display: grid.
  • La unidad fr es nuestra mejor amiga.
  • Usamos gap para dar espacio fácilmente.
